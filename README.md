# Electric-Vehicle-Tableau-Dashboard
Interactive Tableau dashboard analyzing Electric Vehicle Population Data to uncover EV adoption trends, manufacturer performance, vehicle types, and geographic distribution through data-driven visualizations and insights.

## Project Overview

This project presents an interactive Tableau dashboard built using Electric Vehicle Population Data. The dashboard provides insights into electric vehicle adoption trends, manufacturer performance, vehicle types, and geographic distribution to support data-driven decision-making.

## Problem Statement

With the increasing adoption of electric vehicles (EVs), it is important to understand market trends, vehicle distribution, and manufacturer performance. This project aims to analyze EV population data and visualize key metrics through an interactive Tableau dashboard.

## Dataset Information

* Dataset: Electric Vehicle Population Data
* Format: CSV
* Records: EV registration and population information
* Key Attributes:

  * Vehicle Make
  * Model
  * Model Year
  * Electric Range
  * County
  * State
  * EV Type
  * CAFV Eligibility

## Tools Used

* Tableau Public/Desktop
* Microsoft Excel
* CSV Dataset

## Dashboard Features

* Total Electric Vehicles Overview
* EV Adoption by Year
* Top Vehicle Manufacturers
* Electric Vehicle Type Distribution
* Geographic Analysis by County
* Electric Range Analysis
* Interactive Filters and Drill-Downs

## Key Insights

* Identified leading EV manufacturers.
* Analyzed growth trends in electric vehicle adoption.
* Examined regional distribution of EVs.
* Compared battery electric vehicles and plug-in hybrid vehicles.
* Evaluated electric range patterns across different models.
